# Aura – Blind & Low Vision Assistant

**AI navigation assistant for visually impaired users using Gemini 3**

---

## Overview
Aura is an intelligent AI assistant designed to help visually impaired users navigate safely and independently. Using the **multimodal capabilities of Gemini 3**, Aura can detect objects, signs, obstacles, and human gestures in real-time, providing audio guidance and contextual advice to the user.

---

## Features
- **Real-time object and obstacle detection** using Gemini 3’s multimodal reasoning
- **Context-aware navigation**: interprets traffic signals, moving vehicles, and pathways
- **Natural language interaction**: users can ask questions like “What’s in front of me?” or “Is it safe to cross?”
- **Low-latency guidance**: instant audio/text feedback for safety-critical scenarios
- **Continuous learning**: improves detection accuracy over time
